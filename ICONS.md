# Icon Files Required

The extension requires three icon files for the Chrome extension:

- `icon16.png` - 16x16 pixels (toolbar icon)
- `icon48.png` - 48x48 pixels (extension management page)
- `icon128.png` - 128x128 pixels (Chrome Web Store)

## Creating Icons

You can create simple placeholder icons using any image editor:

1. **Using Online Tools**:
   - Visit https://www.favicon-generator.org/ or similar
   - Upload a WhatsApp-themed image or create a simple design
   - Download in multiple sizes

2. **Using Image Editors**:
   - Create a 128x128 pixel image with a WhatsApp green background (#25D366)
   - Add a simple checkmark or "CI" text
   - Export as PNG
   - Resize to create 16x16 and 48x48 versions

3. **Quick Placeholder**:
   - Create a simple green square with white text "CI" (Check-In)
   - Use WhatsApp green: #25D366
   - Use white text: #FFFFFF

## Temporary Solution

If you don't have icons yet, you can:
1. Create simple colored squares as placeholders
2. The extension will work without icons, but Chrome will show a default puzzle piece icon
3. You can add proper icons later

## Icon Specifications

- Format: PNG
- Background: Transparent or solid color
- Recommended: WhatsApp green (#25D366) with white or dark green accents
- Style: Simple, recognizable, professional
