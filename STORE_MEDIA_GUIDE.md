# Chrome Web Store Media Requirements Guide

Based on the screenshot, here's what you need to upload:

## Required Media

### 1. Screenshots (REQUIRED - At least 1, up to 5)

**Requirements:**
- **Size**: 1280x800 OR 640x400 pixels
- **Format**: JPEG or 24-bit PNG (no alpha/transparency)
- **Quantity**: At least 1, maximum 5
- **Purpose**: Show your extension in action

**What to Screenshot:**

#### Screenshot 1: Main Interface (Required)
- Open WhatsApp Web in Chrome
- Open the extension popup
- Show the popup with "Enable Logging" toggle ON
- Show the status indicator (green dot)
- **How to capture:**
  1. Open WhatsApp Web
  2. Click extension icon
  3. Take screenshot of the popup (use Windows Snipping Tool or Chrome DevTools)
  4. Resize to 1280x800 or 640x400

#### Screenshot 2: Events Display (Recommended)
- Show the popup with logged events visible
- Display multiple check-in/check-out events
- Show member names, messages, and timestamps

#### Screenshot 3: CSV Export (Recommended)
- Show a CSV file opened in Excel/Google Sheets
- Display sample check-in/check-out data
- Show the structured format

#### Screenshot 4: Keywords Editor (Optional)
- Show the keywords editing modal
- Display the customizable keywords feature

#### Screenshot 5: Settings View (Optional)
- Show the popup with all settings visible
- Display auto-download toggle and other options

**How to Take Screenshots:**

**Method 1: Windows Snipping Tool**
1. Press `Windows + Shift + S`
2. Select the area to capture
3. Save as PNG
4. Resize using Paint or online tool to 1280x800

**Method 2: Chrome DevTools**
1. Right-click extension popup ‚Üí Inspect
2. In DevTools, click device toolbar icon
3. Set custom size: 1280x800
4. Take screenshot

**Method 3: Browser Extension**
1. Install a screenshot extension
2. Capture the popup
3. Resize if needed

## Quick Creation Guide

### For Screenshots:

1. **Prepare WhatsApp Web:**
   - Open WhatsApp Web
   - Join or create a test group
   - Send some test messages with "check-in" and "check out"

2. **Capture Screenshots:**
   - Screenshot 1: Extension popup with logging enabled
   - Screenshot 2: Popup showing logged events
   - Screenshot 3: CSV file in spreadsheet

3. **Edit Screenshots:**
   - Use Paint, GIMP, or online editor
   - Resize to 1280x800 (or 640x400)
   - Save as PNG or JPEG
   - Remove any transparency (alpha channel)

### For Promo Tiles:

**Option 1: Use Online Tools**
- Canva.com (free templates)
- Figma (free design tool)
- GIMP (free image editor)

**Option 2: Use PowerPoint/Google Slides**
- Create slide with 440x280 or 1400x560 dimensions
- Add text and colors
- Export as image

**Option 3: Use the HTML Tool**
- Open `create-promo-images.html` in browser
- Generate placeholder promo tiles
- Download and use

## Tips

1. **Screenshots are REQUIRED** - You must upload at least 1 screenshot
2. **Quality matters** - Use clear, high-quality images
3. **Show value** - Screenshots should clearly show what the extension does
4. **No transparency** - Make sure PNG files don't have alpha channel
5. **Consistent style** - Use similar colors and style across all images
6. **WhatsApp green** - Use #25D366 for branding consistency
