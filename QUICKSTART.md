# Quick Start Guide

## Installation (5 minutes)

1. **Download/Clone** this repository to your computer

2. **Create Icons** (optional but recommended):
   - Open `create-icons.html` in your browser
   - Click "Download All Icons"
   - Save the three icon files in the extension folder

3. **Load Extension in Chrome**:
   - Open Chrome and go to `chrome://extensions/`
   - Enable "Developer mode" (top-right toggle)
   - Click "Load unpacked"
   - Select the extension folder

4. **Test the Extension**:
   - Open WhatsApp Web (`https://web.whatsapp.com`)
   - Click the extension icon in Chrome toolbar
   - Toggle "Enable Logging" to ON
   - Send a test message with "check-in" or "check out" in a group chat
   - Check the popup to see if the event was logged

## First Use Checklist

- [ ] Extension loaded in Chrome
- [ ] Icons created (or using default)
- [ ] WhatsApp Web opened
- [ ] Logging enabled in popup
- [ ] Test message sent
- [ ] Event appears in popup
- [ ] CSV export works

- Icons missing? Extension still works, just shows default icon

**"Not detecting messages"**
- Make sure you're in a **group chat** (not individual)
- Verify logging is enabled (green dot in popup)
- Refresh WhatsApp Web page after enabling

**"CSV download not working"**
- Check Chrome download permissions
- Make sure you have events logged (check popup)
